# Resonance

**Resonance** は、1枚の写真とその場の録音を一緒に残し、あとから写真・音・場所・時間帯・メモで見返すための iOS アプリです。

現在の main ブランチでは **3Dスキャン機能は提供しません**。体験の中心は、写真撮影、環境音の録音、撮影直後のプレビュー、保存後の再生です。

## 現在の機能

### 記録
- ライブカメラから写真を撮影
- 写真に環境音を重ねて記録
- 録音時間を 3〜20 秒で調整
- シャッター音を避ける遅延録音
- 通常の 1 シーン撮影
- 一定間隔で複数枚を残す Interval capture
- Interval capture の一時停止、再開、終了
- 撮影中の波形表示、残り秒数表示、進行表示
- カメラ、マイク、位置情報の準備状態表示

### 撮影直後プレビュー
- 写真と録音の全画面プレビュー
- 録音波形と再生位置の表示
- 撮り直し
- タイトル入力
- メモ入力
- 場所と撮影日時の確認
- AIキャプション生成
- キャプション文体の切り替え
- Apple Intelligence 対応環境でのキャプション再生成
- 保存前の録音リカバリー状態表示

### 保存と解析
- SwiftData によるローカル保存
- 写真ファイル保存
- 音声ファイル保存
- 解析用音声ファイル保存
- true spatial audio 対応端末での空間オーディオ録音
- stereo fallback
- 音声プロファイル解析
- 波形フィンガープリント生成
- 音量範囲の記録
- 音の方向ホットスポット解析
- 画像タグ、音声タグ、ムード推定
- Speech による文字起こし
- 位置、高度、方角、気圧、端末姿勢などのメタデータ保存

### ライブラリ
- 保存記録の一覧表示
- リスト表示とグリッド表示
- 地図表示
- 日付フィルター
- 時間帯フィルター
- ムードフィルター
- お気に入りフィルター
- 音声付きフィルター
- 新しい順、古い順の並び替え
- 複数選択
- 一括削除
- コレクション追加
- 手動コレクション
- スマートコレクション
- 連続撮影シーンの管理
- Time Capsule
- Nearby memories

### 詳細表示
- 写真の大きなプレビュー
- 録音の再生、一時停止、シーク
- 波形表示
- 写真と録音の全画面プレビュー
- お気に入り切り替え
- 共有
- 編集
- 削除
- メモ表示
- キャプション文体変更
- キャプション再生成
- 記録メタデータ表示
- 文字起こし表示
- 自動タグ表示
- 関連する記録の表示

### 検索
- 音、場所、空気感による検索
- 検索候補表示
- お気に入り絞り込み
- ムードコンパス検索
- 似た空気の記録表示
- コレクション検索
- シーン検索
- 一致理由の表示

### 没入プレビュー
- 写真を全画面表示
- 環境音のループ再生
- 波形表示
- 音量連動の視覚演出
- 音のホットスポット表示
- 端末の向きとドラッグによる音場調整
- Reduce Motion 対応

### 設定
- Time Capsule 表示切り替え
- Nearby memories 表示切り替え
- Nearby memories 検索半径
- 没入プレビューの環境粒子
- 音量連動ライト
- 音のホットスポット表示
- 端末の向きによる音場調整
- スライドショー自動再生
- スライドショー切替間隔
- 既定のプレビュー音量
- 既定のキャプション文体
- Interval capture の既定値

## 廃止した機能

- 点群生成

- 3Dスキャンプレビュー

## 開発メモ

このリポジトリには、Windows から編集しつつ GitHub Actions で IPA を生成するための構成も含まれています。

ビルド方法、署名、IPA 出力、Sideloadly での導入手順は `BUILD_MEMO.md` にまとめています。
