# Resonance

**Resonance** は、写真だけでなく **その場の空気や音の余韻まで一緒に保存する** ためのカメラアプリです。

シャッターを押すと、1枚の写真と環境音をひとつの記録として残します。あとから見返すときは、画像・音・場所・時間帯・空間センサーの気配が重なり、ただのアルバムではなく **「その瞬間に戻る体験」** を目指しています。

## このアプリが大切にしていること

1. **写真ではなく、場の記憶を残す**
2. **撮るときはカメラらしく直感的である**
3. **見返すときは空気感まで感じられる**

## 主な体験

### 1. Camera-first capture
- ライブカメラを全面に出した撮影画面
- 起動直後は “Preparing your scene” で心地よく待てる導入
- シャッターひとつで写真 + 可変長の環境音を記録
- 撮影中も「いま何が起きているか」が分かる明快な UI
- 録音秒数をその場の空気に合わせて変更可能

### 2. Memory Scene
- 撮影直後に写真と音をまとめて確認できるフルスクリーンレビュー
- 見切れないプレビュー表示
- 波形を見ながら、その場の空気を再生
- タイトルやメモを追加して保存
- 明るい写真でも暗い写真でも読みやすい多層プレビュー

### 3. Atmosphere metadata
- 撮影時刻から導く時間帯のムード
- 場所ラベル
- 波形フィンガープリント
- 位置・高度・方角・気圧・相対高度・端末姿勢などの空間スナップショット
- 自動タグ、音声文字起こし、ムード推定

### 4. Revisit experience
- ライブラリで写真と音の“気配”を一覧
- 一覧 / 地図 を切り替えて場所から記憶を選択
- 時期・時間帯・お気に入り・音声有無・ムードで絞り込み
- 詳細画面で、場所・音・時間帯をまとめて再訪
- タイトル / メモ / タグ / ムード / 場所で横断検索

## こんなときに向いています

- 旅先の「空気」を残したい
- 子どもの声や生活音ごと記録したい
- 雨、街、海、カフェなどの音の記憶を集めたい
- ただの写真管理より、感情や場の温度まで残したい

## 現在の実装内容

- SwiftUI ベースの iOS アプリ
- `SwiftData` によるローカル保存
- 写真 + 音声ファイル保存
- 地図ブラウズとフィルター
- 空間センサー情報の保存
- `Vision` / `Speech` / 軽量音声解析による自動タグ付け
- Memory Scene レビュー
- ライブラリ / 詳細 / 検索
- ダークモード対応の atmospheric UI
