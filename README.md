# Resonance

**Resonance** は、1枚の写真とその場の録音を一緒に残し、あとから写真・音・場所・時間帯・メモで見返すための iOS アプリです。

現在の main ブランチでは **3Dスキャン機能は提供しません**。体験の中心は、写真撮影、環境音の録音、撮影直後のプレビュー、保存後の再生です。

## 製品版アップグレード方針

- 写真、環境音、場所、時間帯、気配をひとつの「記憶」として扱う
- クラウドAIに依存せず、Vision、Speech、FoundationModels、音声解析をオンデバイス中心で使う
- 既存の記録を維持しながら SwiftData V3 と追加メタデータで段階的に品質を上げる
- 画面装飾よりも写真、波形、地図、記録データを主役にする

## 現在の機能

### 記録
- ライブカメラから写真を撮影
- 写真に環境音を重ねて記録
- 録音時間を 3〜20 秒で調整
- シャッター音を避ける遅延録音
- 通常の 1 シーン撮影
- 一定間隔で複数枚を残す Interval capture
- Interval capture の一時停止、再開、終了
- 撮影中の波形表示、残り秒数表示、進行表示
- 録音品質メーター
- 入力ピーク警告
- 静かすぎる録音の警告
- カメラ、マイク、位置情報の準備状態表示

### 撮影直後プレビュー
- 写真と録音の全画面プレビュー
- 録音波形と再生位置の表示
- 波形ドラッグによるシーク
- ピンチズームとドラッグ視差
- 撮り直し
- タイトル入力
- メモ入力
- 場所と撮影日時の確認
- AIキャプション生成
- キャプション文体の切り替え
- Apple Intelligence 対応環境でのキャプション再生成
- 保存前の録音リカバリー状態表示

### 保存と解析
- SwiftData によるローカル保存
- 写真ファイル保存
- 音声ファイル保存
- 解析用音声ファイル保存
- true spatial audio 対応端末での空間オーディオ録音
- stereo fallback
- 音声プロファイル解析
- 波形フィンガープリント生成
- 音量範囲の記録
- 音の方向ホットスポット解析
- 写真の主題、明るさ、色傾向、サリエンシー解析
- dominant colors と hero focus point の保存
- 記録品質スコア、音声品質警告、検索キーワードの保存
- 生成タイトルの保存
- 画像タグ、音声タグ、ムード推定
- Speech による文字起こし
- 位置、高度、方角、気圧、端末姿勢などのメタデータ保存

## データ形式

- SwiftData V3
- 既存 V1 / V2 記録の軽量移行
- 写真、音声、解析用音声はローカル保存
- 追加解析情報は `MemoryAtmosphereMetadata` JSON に保存
- 欠損した解析情報はライブラリまたは詳細表示時にバックフィル

## 廃止した機能

- 3Dスキャン撮影
- 3Dスキャン最適化
- 点群生成
- 3Dスキャンデータの保存
- 3Dスキャンプレビュー
